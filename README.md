# Physics Simulation Prompt Toolkit 🔬

An open-source collection of AI prompts and tools to help 
educators explain physics concepts using simulations.

## What's Inside
- Prompt templates for physics simulations
- Proton-proton chain reaction simulation prompts
- Nuclear fusion concept explanations
- Mechanics, optics, and space-time prompts
- Prompt engineering patterns for educators

## Who Is This For
- Teachers and tutors
- Students learning physics
- Developers building educational AI tools

## Why This Project
Physics concepts become clearer when visualized through 
simulations. This toolkit helps educators use AI to create 
accurate, interactive explanations for students.

Pull requests welcome! Open an issue to suggest new
